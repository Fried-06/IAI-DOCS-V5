Dans le crypto-système RSA, Awa choisit les paramètres ci-après. Dans chaque cas, dire si le choix est bien fait.
1. $p = 11$, $q = 29$, $e = 3$,
2. $p = 33$, $q = 83$, $e = 3$,

### Exercice 2. (7 pts)

Dans le système RSA, Awa choisit ($p = 7$, $q = 37$ et $e = 5$) et Baba choisit ($p = 17$, $q = 41$, $e = 3$).

1. Déterminer la clé privée d'Awa.
2. Baba souhaite envoyer le message $m = 13$ à Awa. Déterminer le message codé $c$.
3. Baba signe le message $m = 15$, code le message et sa signature avant de les envoyer à Awa.
   (a) Quel est le cryptogramme reçu par Awa ?  
   (b) Comment Awa décode-t-elle le message reçu ?

### Exercice 3. (4.5 pts)

1. Dans le système RSA, montrer que si Awa et Baba utilisent le même module $n$ et des exposants $e_A$ et $e_B$ qui sont premiers entre eux, alors une troisième personne peut lire un même message $m$ codé et envoyé à Awa et Baba.
2. Dans le système RSA, montrer que si un utilisateur choisit $p$ et $q$ suffisamment proches, il est possible de factoriser son module $n$. Application : factoriser $n = 204223$.
