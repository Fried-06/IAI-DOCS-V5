## Exercice — Définir une classe `Animal` en Python avec héritage *(8 pts)*

Un parc animalier souhaite développer un petit programme en Python permettant de représenter
différents types d'animaux selon leur régime alimentaire.

> Vous devez concevoir ce programme en utilisant la **Programmation Orientée Objet (POO)**
> et la notion d'**héritage simple**.

**Travail demandé** : écrire le code Python permettant de créer :

- Une **classe mère** : `Animal`
- Deux **classes filles** : `Carnivore` et `Herbivore`

---

### Partie 1 — Classe mère `Animal` *(2 pts)*

La classe `Animal` doit contenir :

**Attributs**

| Attribut | Description          |
|----------|----------------------|
| `nom`    | Nom de l'animal      |
| `age`    | Âge de l'animal      |
| `poids`  | Poids de l'animal    |

**Méthodes**

1. **`__init__(self, nom, age, poids)`** — Constructeur initialisant les trois attributs.

2. **`se_deplacer()`** — Affiche :

   ```
   L'animal se déplace.
   ```

3. **`manger()`** — Affiche :

   ```
   L'animal mange.
   ```

4. **`__str__()`** — Retourne les informations de l'animal (nom et âge).

---


### Partie 3 — Classe fille `Herbivore` *(2 pts)*

### Correction de la classe mère `Personne` *(1,5 pt)*

Le code ci-dessous contient une erreur. **Identifiez et corrigez-la.**

```python
# Classe mère — contient une erreur, corrigez-la (1,5 pt)
class Personne:
    def __init__(self, nom, age):       # ← erreur à corriger ici
        self.nom = nom
        self.age = age

    def se_presenter(self):
        print(f"Je m'appelle {self.nom} et j'ai {self.age} ans.")
```

---

### Classe fille `Employe` — analyse de code

```python
class Employe(Personne):
    def __init__(self, nom, age, poste, salaire):
        super().__init__(nom, age)   # ← Q: que fait super().__init__ ? (1 pt)
        self.poste   = poste
        self.salaire = salaire

    def se_presenter(self):          # ← Q: expliquer après correction (1 pt)
        print(f"Je m'appelle {self.nom}, j'ai {self.age} ans "
              f"et je suis {self.poste} avec un salaire de {self.salaire} FCFA.")
```

**Questions**

1. *(1 pt)* Que fait le mot-clé `super().__init__` dans le constructeur de `Employe` ?

2. *(1 pt)* Expliquez ce que fait la méthode `se_presenter()` de `Employe` après correction.
