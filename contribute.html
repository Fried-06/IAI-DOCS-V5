🤖 Comment utiliser l'Intelligence Artificielle ?

Si vous utilisez ChatGPT, Claude ou Gemini pour convertir un document, fournissez-lui un document dans lequel le texte est sélectionnable (PDF OCR ou Word) et non une image scannée. Copiez le prompt exact ci-dessous et collez-le avec votre document pour garantir la compatibilité avec notre moteur Sphinx.

Convertis ce document en Markdown strict pour Sphinx/MyST. Règles absolues : 1) Un seul titre H1 (`#`) pour le nom de l'épreuve tout en haut. 2) Utilise `##` (H2) uniquement pour les grandes sections (ex: EXERCICE 1). Ne JAMAIS utiliser de `###` ou `####`. 3) Numéros de questions : Utilise uniquement du texte en gras (ex: **1)** ou **a)**) sur la même ligne que la question. Le barème doit être en bout de ligne. 4) Mathématiques : Les équations en ligne doivent utiliser `$ x=2 $` et les blocs isolés doivent utiliser `$$`. N'utilise jamais `\(` ou `\[`. 5) Reproduis les tableaux en format table Markdown standard. 6) Remplace tout graphe ou image non convertible par "[Image/Graphe manquant]".
